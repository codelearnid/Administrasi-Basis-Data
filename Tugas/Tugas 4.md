# **Tugas Basic Python**

## **Instruksi Umum**

1. Gunakan Python versi 3.x.
2. Tuliskan semua jawaban dalam file `.py` atau di notebook Jupyter.
3. Sertakan komentar (`#`) untuk menjelaskan setiap langkah kode yang dibuat.
4. Kerjakan semua soal, tugas dinilai berdasarkan kelengkapan, kebenaran logika, dan kerapihan kode.

---

## **Soal 1: Variabel dan Tipe Data**

1. Buat variabel `nama` yang menyimpan nama kamu.
2. Buat variabel `umur` yang menyimpan umur kamu (integer).
3. Buat variabel `tinggi` yang menyimpan tinggi badan kamu dalam meter (float).
4. Buat variabel `is_student` yang menyatakan apakah kamu seorang mahasiswa (boolean).
5. Tampilkan semua variabel tersebut dengan format:

   ```
   Nama: <nama>, Umur: <umur> tahun, Tinggi: <tinggi> m, Mahasiswa: <is_student>
   ```

---

## **Soal 2: Operasi Matematika**

1. Buat dua variabel `a = 15` dan `b = 4`.
2. Hitung hasil penjumlahan, pengurangan, perkalian, pembagian, modulus, dan pangkat dari kedua variabel.
3. Tampilkan hasilnya dalam format yang jelas, misalnya:

   ```
   15 + 4 = 19
   ```

---

## **Soal 3: Struktur Kontrol**

1. Minta input dari user untuk memasukkan sebuah angka (`angka`).
2. Buat kondisi:

   * Jika angka > 0 → tampilkan “Angka positif”
   * Jika angka < 0 → tampilkan “Angka negatif”
   * Jika angka = 0 → tampilkan “Angka nol”

---

## **Soal 4: Perulangan**

1. Buat sebuah list `buah = ["apel", "jeruk", "pisang", "mangga"]`.

2. Gunakan perulangan `for` untuk menampilkan semua buah dengan format:

   ```
   Buah ke-1: apel
   Buah ke-2: jeruk
   ...
   ```

3. Gunakan perulangan `while` untuk mencetak angka dari 1 sampai 10.

---

## **Soal 5: Fungsi**

1. Buat fungsi `luas_persegi_panjang(p, l)` yang menerima panjang `p` dan lebar `l`, lalu mengembalikan luasnya.
2. Buat fungsi `keliling_persegi_panjang(p, l)` yang mengembalikan kelilingnya.
3. Gunakan fungsi tersebut untuk menghitung luas dan keliling persegi panjang dengan panjang = 10 dan lebar = 5.

---

## **Soal 6: List dan Manipulasi Data**

1. Buat list `nilai = [70, 85, 60, 90, 75]`.
2. Tambahkan nilai baru `95` ke dalam list.
3. Urutkan list dari nilai terkecil ke terbesar.
4. Hitung rata-rata nilai.
5. Tampilkan hasilnya.
